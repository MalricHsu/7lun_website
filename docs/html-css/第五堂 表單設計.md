### 三、Bootstrap 為例

1. Bootstrap 結構概念
   - [https://github.com/twbs/bootstrap/blob/main/scss/bootstrap.scss](https://github.com/twbs/bootstrap/blob/main/scss/bootstrap.scss)
     - CSS Normalize
     - Content
     - Grid System
     - **Component**
     - **Helpers**
     - **Utilities**
2. Bootstrap 元件運用概念

- OOCSS
  1. 容器與內容分離
  2. 結構與樣式分離
     範例1：

     ```html
     <button type="button" class="btn btn-primary">Primary</button>
     ```

  範例2 - 額外樣式：

        ```html
        <button type="button" class="btn btn-primary btn-lg">Large button</button>
        ```

  範例3 - 狀態：

        ```html
        <button type="button" class="btn btn-primary" disabled>Primary button</button>
        ```

- 技巧：
  - 結構
  - 樣式

1. 手寫 SCSS 結構參考
   - 手刻

     ```
     @import "variable";// 變數
     @import "reset";  // reset.css

     // 1. 元件(進階)*
     @import "button";
     @import "...";

     // 2. 雜項(進階)*
     @import "utilities";

     // 3. 共用樣式（如有 Bootstrap 通常不會寫這段）
     @import "layout"; // 共同框架、第一層

     // 4. 分頁樣式（可在此覆蓋 Bootstrap 樣式）
     @import "index";
     @import "page1";

     ```

   - Bootstrap

     ```
     @import "variable"; // 變數
     @import "bootstrap";

     // 1. 自定義元件(進階)*
     @import "button";
     @import "...";

     // 2. 雜項(進階)*
     @import "utilities";

     // 3. 分頁樣式（可在此覆蓋 Bootstrap 樣式）
     @import "index";
     @import "page1";
     ```
